## Overview
Comprehensive dataset of key health indicators for Venezuela (2020-2025). Includes mental health metrics (depression, anxiety, suicide rates), healthcare workforce decline, infectious disease burden (malaria, dengue, HIV, measles), non-communicable disease prevalence, healthcare system capacity, and migration-related health impacts. Sources: PAHO, WHO, World Bank, ACNUR/UNHCR, IOM, UNAIDS.
